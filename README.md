# api-rest-express
## Installation

Install [Node.js](https://nodejs.org/es/download).

Install the dependencies and start the server.

```sh
npm i
node src/index.js
```

Go to documentation [http://localhost:3000/api-docs/](http://localhost:3000/api-docs/)
